# Sanjarbek Otabekov — Full-Stack Developer Portfolio

![Sanjarbek Otabekov Portfolio Preview](public/preview.jpg)

Bu zamonaviy, tezkor va yuqori unumdorlikka ega shaxsiy portfolio loyihasi bo'lib, **Next.js 14**, **Tailwind CSS** va **Firebase** texnologiyalari asosida yaratilgan. Loyiha nafaqat portfolio vazifasini bajaradi, balki to'liq interaktiv **Admin Panel**, **PWA (Progressive Web App)** imkoniyatlari va maksimal **SEO optimizatsiyasi** bilan ta'minlangan.

## 🚀 Texnologiyalar (Tech Stack)

Loyihaning asosi quyidagi zamonaviy texnologiyalardan tashkil topgan:

- **Frontend:** Next.js 14 (App Router), React 18, TypeScript
- **Styling:** Tailwind CSS, Framer Motion (animatsiyalar uchun), Lucide React (ikonkalar)
- **Backend & Database:** Firebase (Authentication, Firestore, Storage)
- **SEO & Optimizatsiya:** Metadata API, JSON-LD Structured Data, WebP/AVIF Image Optimization
- **Arxitektura:** PWA, Server-Side Rendering (SSR) & Static Site Generation (SSG)

## ✨ Asosiy Imkoniyatlar (Features)

- 🎨 **Premium UI/UX Dizayn:** Dark va Light rejimlariga ega, juda chiroyli glassmorphism uslubidagi interfeys.
- 📱 **PWA Qo'llab-quvvatlashi:** Ilovani bevosita telefon yoki kompyuterga o'rnatish imkoniyati (Manifest va Service Worker orqali).
- 📈 **Maksimal SEO:** Qidiruv tizimlari (Google, Yandex) uchun maksimal optimizatsiya (Sitemap, Robots.txt, JSON-LD, OpenGraph va Twitter kartalari).
- 🔒 **Xavfsiz Admin Panel:** Loyihalar, sertifikatlar va sayt statistikasini jonli tarzda boshqarish uchun Firebase bilan himoyalangan admin boshqaruv paneli (`/admin`).
- ⚡ **Tezkor Ishlash:** Vercel va Next.js kombinatsiyasi yordamida eng tezkor yuklanish tezligi.

## 🛠️ O'rnatish va Ishga Tushirish

Loyihani o'z kompyuteringizda ishga tushirish uchun quyidagi qadamlarni bajaring:

### 1. Repozitoriyni yuklab olish
```bash
git clone https://github.com/sanjarbek404/portfolio.git
cd portfolio
```

### 2. Kutubxonalarni o'rnatish
```bash
npm install
# yoki
yarn install
```

### 3. Muhit o'zgaruvchilari (Environment Variables)
Loyihaning asosiy papkasida `.env.local` faylini yarating va Firebase sozlalamalarini kiriting:

```env
NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
```

### 4. Mahalliy serverni ishga tushirish
```bash
npm run dev
# yoki
yarn dev
```
Sayt `http://localhost:3000` manzilida ishga tushadi.

## 🚀 Vercel'ga Deploy Qilish (Deployment)

Loyiha Vercel platformasi uchun to'liq optimizatsiya qilingan.

1. Loyihani GitHub ga joylashtiring.
2. Vercel dashboardiga kiring va "Add New Project" tugmasini bosing.
3. GitHub repozitoriyni tanlang va Import qiling.
4. **Environment Variables** qismida barcha `.env.local` ichidagi Firebase kalitlarini qo'shing.
5. Deploy tugmasini bosing.
6. Sozlamalardan (Settings -> Domains) o'zingizning shaxsiy domeningizni (`sanjarme.uz`) ulab qo'ying.
